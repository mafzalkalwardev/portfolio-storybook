# Portfolio Storybook

A whimsical, novel-reader aesthetic portfolio for **Muhammad Afzal Kalwar** — Full-Stack Developer & Automation Engineer.

Soft pastels, storybook chapters, and gentle animations present the same projects, skills, and contact details as the main portfolio in a cozy, literary style.

## Live

Deploy via GitHub Pages from the `main` branch.

## Structure

- **Chapter 1 · Prologue** — Hero, stats, education
- **Chapter 2 · The Author's Note** — Bio, timeline, services
- **Chapter 3 · Tales I've Written** — 38 projects with filters
- **Chapter 4 · Tools of the Trade** — Skills & tech stack
- **Chapter 5 · Send a Letter** — Contact form

## Tech

Static HTML, CSS, and vanilla JavaScript — no build step required.

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```
